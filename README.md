# GETSTUFF — Интернет-магазин строительного крепежа

Полнофункциональный прототип интернет-магазина для GETSTUFF (getstuff.ru).

## Запуск

Откройте файл `index.html` в браузере или запустите локальный сервер:

```bash
# Python
python -m http.server 8080

# Node.js (если установлен)
npx serve .
```

Сайт будет доступен по адресу `http://localhost:8080`

## Структура

```
getstuff/
├── index.html          — Главная страница
├── catalog.html        — Каталог с фильтрами
├── product.html        — Карточка товара
├── cart.html           — Корзина
├── checkout.html       — Оформление заказа
├── account.html        — Личный кабинет
├── favorites.html      — Избранное
├── css/style.css       — Стили (зелёная палитра)
└── js/
    ├── data.js         — Каталог товаров (20 позиций, 6 категорий)
    ├── app.js          — Корзина, поиск, избранное
    └── components.js   — UI-компоненты и страницы
```

## Реализованный функционал

- **Каталог** — 6 категорий, 20 товаров с характеристиками
- **Фильтрация** — по категории, материалу, диаметру, цене, наличию
- **Поиск** — автоподстановка по артикулу и названию
- **Корзина** — добавление, изменение количества, localStorage
- **Оформление заказа** — форма с оплатой ЮKassa / CloudPayments
- **Личный кабинет** — история заказов, бонусная система
- **Избранное** — сохранение товаров
- **Адаптивный дизайн** — mobile-first с нижней навигацией

## Дизайн

- Зелёная палитра (#2D6A3E, #00A650, #1A472A)
- Mobile-first (референс: ПОРЯДОК)
- Desktop-адаптация для планшетов и ПК
- Брендинг GETSTUFF (референс: ChatGPT-макет)
